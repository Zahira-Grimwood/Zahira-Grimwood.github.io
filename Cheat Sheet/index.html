How to use Colours in CSS...

Hexadecimal or Hex Colours

What is it?

Hexadecimal (Hex) is a number system that has sixteen digits, 0 to 9 followed by “A” to “F”. Hex values always begin with # and specify values of red, blue, and green.

Example...

An example of a Hex color representation is #123456. 12 is red, 34 is Green, and 56 is Blue and together the values make blue.

RGB

What is it?

RGB colors use the rgb() syntax with one value for red, one value for blue and one value for green. RGB values range from 0 to 255.

Example...

An example of an RGB value is rgb(255, 0, 0) which makes red.

HSL

What is it?

HSL stands for hue (the color itself), saturation (the intensity of the color), and lightness (how light or dark a color is). Hue ranges from 0 to 360 and saturation and lightness are both represented by percentages.

Example...

An example of an HSL value is hsl(74, 73%, 51%) which is green.
